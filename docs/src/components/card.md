# Card

`ui-card` is a simple content wrapper with a fixed aspect ratio and border radius.

## Basic Usage

```vue
<ui-card>Example</ui-card>
```

## Sizes

```vue
<ui-card size="large">Large</ui-card>
<ui-card size="base">Base</ui-card>
<ui-card size="small">Small</ui-card>
<ui-card size="xs" />
```

## Props

| Prop   | Type   | Default | Description                       |
| ------ | ------ | ------- | --------------------------------- |
| `size` | String | `base`  | `large`, `base`, `small`, or `xs` |
